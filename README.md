# Проект: Место

### Описание

Проект, где присутствует профиль пользователя, и есть возможность его редактирования. Измение профиля происходит черер
вызов попапа с полями для редактируемых данных.

### Технологии

Git, JavaScript, СSS, HTML, БЭМ.
